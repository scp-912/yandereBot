# yandereBot

从 yandere 爬图的 bot

## 简介

**yandereBot** 是一个用来从 [Yandere](https://yande.re) 网站爬取图片的机器人。主要使用 Python 编写。

## 安装

要安装所需的依赖项，请运行以下命令：

```bash
pip install -r requirements.txt
```bash
